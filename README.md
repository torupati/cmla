# CMLA: Classical Machine Learning Algorithms

[![Version](https://img.shields.io/badge/version-v0.1.0--alpha-orange)](https://github.com/torupati/cmla/releases/tag/v0.1.0-alpha)
[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![Documentation](https://img.shields.io/badge/docs-GitHub%20Pages-green)](https://torupati.github.io/cmla/)

**CMLA** is a Python package for mathematical algorithms and data analysis,
providing implementations of fundamental machine learning and statistical algorithms
for educational and research purposes.

## Version Information

**Current Version: v0.1.0-alpha** (Initial Alpha Release)

This is an early alpha release containing:
- Core implementations of K-means, GMM, and HMM algorithms
- Command-line interfaces for all algorithms
- Comprehensive documentation and examples
- Utility functions for model parameter manipulation
